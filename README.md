# About
28 Keys Later is a simple game that motivates little children to learn to use a keyboard.

Players defend against a cute zombie horde by typing different letters on the keyboard, smacking zombies away with a cute hammer.

Over ten levels, the waves get progressively more difficult, starting slow with single keys, and ending fast with five letter words.

It is meant to be played together, parent and child, with a bright, colorful keyboard.

# Code
This game is developed in javascript with PixiJS and a few other small libraries. The build system is Electron. Feel free to email bugsby dot carlin at his gmail account for a deeper description.
